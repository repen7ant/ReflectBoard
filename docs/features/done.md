# Страница «Сделанное»

Хронологический список выполненных задач с фильтрацией и поиском.

## Фильтры

- **Поиск:** по заголовку, описанию, рефлексии и тегам (debounce 300 мс). Если запрос начинается с `#` — ищем по тегам; без `#` — по тексту.
- **Фильтр по дате:** пресеты: Сегодня / 7 дней / 30 дней / Всё время + произвольный диапазон. По умолчанию — последние 30 дней.
- **Фильтр по категории:** одиночный выбор из списка.
- Активные фильтры показываются как dismissible-бейджи; есть кнопка «Сбросить всё».

## Карточка в списке

- Заголовок
- Категория (с fallback на snapshot если категория удалена: `activity.category ?? snapshot`)
- Теги
- Дата выполнения, время потраченное, дедлайн (столбиком справа)
- Превью рефлексии (первые 2 строки)
