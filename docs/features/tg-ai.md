# Telegram-бот

Бот служит для пассивного и активного взаимодействия с системой вне браузера.

## Команды

| Команда | Описание |
| ------- | -------- |
| `/start` | Привязать Telegram-аккаунт к ReflectBoard (через deep link с сайта) |
| `/unlink` | Отвязать аккаунт |
| `/add <название>` | Быстро добавить задачу в Backlog |
| `/stats 7d\|30d\|90d\|all` | Статистика за период: выполненные задачи, время, streak, категории |
| `/analyze 7d\|30d\|90d\|all` | AI-анализ статистики (см. ниже) |
| `/settings` | Просмотр и настройка уведомлений |

## Уведомления

**Дедлайны** — срабатывают в 00 и 30 минут каждого часа. Настраиваемые пороги: один или несколько (напр. за 168ч / 72ч / 24ч). Каждый порог уведомляет один раз (Redis dedup).

**Напоминание залогировать активности** — ежедневно в настраиваемое время (`/settings reminder HH:MM`).

**Напоминание об активных задачах** — ежедневно в настраиваемое время (`/settings today HH:MM`), показывает суммарное количество задач в Today + In Process.

## Настройки (`/settings`)

| Ключ | Описание | Пример |
| ---- | -------- | ------ |
| `deadline` | Пороги уведомлений в часах (через пробел) | `/settings deadline 24 72 168` |
| `reminder` | Время напоминания о логировании активностей | `/settings reminder 09:00` |
| `today` | Время напоминания об активных задачах | `/settings today 08:00` |
| `timezone` | UTC-смещение | `/settings timezone +3` |

## AI-анализ

`/analyze <период>` — получает данные через FastAPI (`GET /api/v1/analytics`), формирует промпт и отправляет в AI-провайдер. Rate limit: 1 запрос в час на пользователя (Redis).

Провайдеры (адаптер-паттерн): `anthropic` или `openai_compatible`. Задаётся в `settings.toml`:

```toml
[ai]
provider = "anthropic"
api_key = "sk-ant-..."
```

В промпт передаётся: количество выполненных задач, продуктивность, суммарное время, категории, теги, streak, completion rate.
