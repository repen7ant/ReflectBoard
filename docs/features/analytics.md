# Страница «Статистика»

Роут `/analytics` в Laravel. Исторические данные из MySQL через Eloquent, live-данные из Redis.

## Периоды

Скользящие: 7 дней / 30 дней / 90 дней / всё время. Нет деления «за день» — если сессия пересекает полночь, она не разрывается.

## Визуализации

- **Тепловая карта активности** (GitHub-style) — активность по дням за выбранный период
- **Облако тегов** — самые частые теги
- **Среднее время** от создания до выполнения задачи
- **Распределение времени по категориям** — круговая + столбчатая диаграмма

## Live-данные из Redis

Ключи: `stats:user:{id}:daily:{date}:category:{category_id}`

- Общее количество продуктивных минут за последние 24 часа
- Распределение по категориям (прогресс-бары)
